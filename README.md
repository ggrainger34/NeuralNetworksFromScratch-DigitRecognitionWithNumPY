- Python 3.6+

- matplotlib 3.9.2
- numpy 2.1.3
- pygame 2.6.1

## Module Installation / Setup

```git clone <repo-url>```

```cd <repo-folder>```

```pip install -r requirements.txt```

or

```pip3 install -r requirements.txt```

or

```python - m pip install -r requirements.txt```

or

```python3 -m pip install -r requirements.txt```
